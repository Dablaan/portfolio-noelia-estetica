# [cite_start]PRD: Espacio Digital "Como en Casa" – Noelia Estética [cite: 1]

## 1. Resumen Ejecutivo
[cite_start]Creación de una Landing Page de alta conversión para Noelia, especialista en estética[cite: 3]. [cite_start]El sitio web funcionará como un catálogo digital elegante que centraliza el tráfico de redes sociales y facilita la reserva de citas mediante una vía de contacto directa y profesional (WhatsApp y llamadas)[cite: 4].

## 2. Objetivos del Negocio
* [cite_start]**Profesionalización:** Proyectar una imagen de "detallista, limpia y polifacética"[cite: 6]. [cite_start]Transformar el interés de los usuarios en citas confirmadas mediante un sistema de un solo clic[cite: 6].
* [cite_start]**Filtro de Gestión:** Informar sobre la política de "Solo cita previa" para optimizar la agenda de la profesional[cite: 7].

## 3. Requisitos Funcionales
* [cite_start]**Catálogo de Servicios Estrella:** Sección destacada para Microblading y Micropigmentación[cite: 9].
* [cite_start]**Menú de Estética Completa:** Listado visual que incluye uñas acrílicas, esmalte semipermanente y pedicuras Spa[cite: 10].
* [cite_start]**Botón de Llamada a la Acción (CTA):** Botón flotante para enviar un WhatsApp con el mensaje: "Hola, estoy interesada en el servicio de..."[cite: 11].
* [cite_start]**Acceso Telefónico:** Opción de llamada directa al número 687 43 25 00[cite: 12].
* [cite_start]**Sección de Valor Diferencial:** Bloque de texto que resalte el "ambiente tranquilo", el "trato súper personalizado" y el "cuidar y mimar" a cada cliente[cite: 13, 14].

## 4. Requisitos No Funcionales
* [cite_start]**Identidad Visual:** Uso de tonos rosas pastel y verdes[cite: 16].
* [cite_start]**Estilo:** Diseño que evoca la sensación de estar "como en casa"[cite: 17]. 
* [cite_start]**Disponibilidad:** Optimización total para dispositivos móviles (Mobile-first)[cite: 17].

## 5. Stack Tecnológico Recomendado
* **Plataforma:** Framework Astro y Tailwind CSS (Garantiza rendimiento óptimo, código modular y alineación estricta con las reglas globales del proyecto para un coste cero).
* [cite_start]**Integración:** API de WhatsApp para mensajes predefinidos[cite: 20].
* [cite_start]**Imágenes:** Galería optimizada con fotos reales de trabajos anteriores y el feed de Instagram[cite: 21].

## 6. Fases de Desarrollo
1. [cite_start]**Estructura:** Definición de secciones basadas en la tarjeta de visita[cite: 23].
2. [cite_start]**Diseño:** Implementación de la paleta rosa/verde y tipografías elegantes[cite: 24].
3. [cite_start]**Contenido:** Redacción de las normas de la casa (Avisar si no se puede asistir, solo cita previa)[cite: 25].
4. [cite_start]**Lanzamiento:** Configuración del enlace en la biografía de Instagram[cite: 26].
