# Flujo de Trabajo (Workflow)

- [ ] **Fase 1: Setup Inicial**
  - Instalar proyecto base de Astro.
  - Limpiar archivos innecesarios.
  - Inicializar repositorio Git.
- [ ] **Fase 2: Estructura Base (Arquitectura)**
  - Crear `Layout.astro` (cabecera, pie de página).
  - Configurar paleta de colores en Tailwind.
- [ ] **Fase 3: Desarrollo de Componentes (UI)**
  - Sección Hero (Presentación).
  - Galería de Servicios / Trabajos (Fotos).
  - Botón flotante de WhatsApp.
- [ ] **Fase 4: Integración RRSS**
  - Añadir enlace/botón de Instagram.
- [ ] **Fase 5: Despliegue**
  - Subir código a GitHub.
  - Conectar y desplegar en Vercel.
- [ ] **Fase 6: Auditoría UI/UX y Copywriting**
  - Analizar el impacto visual de los componentes actuales.
  - Proponer mejoras en los textos para aumentar la conversión.
