# Reglas Globales del Proyecto: Portfolio Beauty Salon

## Contexto
- **Objetivo:** Web tipo portfolio para servicios de estética (uñas acrílicas, semipermanente, pedicuras, microblading).
- **Stack Tecnológico:** Astro (Framework) + Tailwind CSS (Estilos). 
- **Presupuesto:** 0€. Solo usar herramientas Open Source y tier gratuito (Vercel para hosting).

## Directrices de Diseño (UI/UX)
- **Mobile-First Estricto:** Diseñar siempre para móvil primero. El 90% del tráfico será móvil.
- **Estética:** Minimalista. Fondos blancos, grises muy claros o tonos *nude*. Tipografía sans-serif elegante y legible.
- **Protagonismo:** Las imágenes del trabajo deben destacar sin distracciones visuales.

## Directrices de Desarrollo
- Escribir código modular: usar componentes de Astro (`.astro`) reutilizables.
- Nada de APIs complejas (ej. Meta API). Para redes, usar enlaces directos o widgets embebidos simples.
- Zona de contacto: Exclusivamente a través de un botón flotante/enlace directo a la API web de WhatsApp (`wa.me`).
- Idioma del código y comentarios: Español.
